# Price Prediction
In the usd_eur.ipynb you can find the whole process from exploring the data to making the predictions.
In the models folder you can find the trained models.
In the data folder you can find the raw data that was used to train the models.

# Project Setup
Clone the project
```sh
git clone https://github.com/CodingToffee/price-prediction.git
```
Install requirements
```sh
pip install -r requirements.txt
```
